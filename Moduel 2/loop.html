What is a Loop?
A loop is a programming tool that repeats a block of code as long as a condition is true (or for a fixed number of times).
1. For Loop
Use a for loop when you know how many times you want to repeat something.
Python Example
for i in range(0, 5):
    print(i) # Output: 0, 1, 2, 3, 4
  • The loop starts at 0 (i = 0)
  • Keeps going as long as i < 5
  • After each print, i is increased by 1
âœ… Perfect for fixed sequences, like printing tables, sums, etc.
2. While Loop
Use a while loop when you donâ€™t know exactly how many times youâ€™ll loop, but you know the condition.
Python Example

while i < 5:
    print(i)
    i += 1 # Output: 0, 1, 2, 3, 4
  • i starts at 0
  • If i < 5, it prints and increases i
  • When i becomes 5, the loop exits
âœ… Great for waiting for user input, checking sensor values, or logging into a system.
3. Simulating Do-While Loop in Python
Python does not have a built-in do-while loop, but you can simulate it using while True and break.
Python Example
i = 0
while True:
    print(i)
    i += 1
    if i >= 5:
        break # Output: 0, 1, 2, 3, 4
  • The code inside the loop runs at least once
  • Breaks out of the loop when i >= 5
âœ… Useful for input validation, menu loops, and games.
Flowchart: While vs Do-While Loop
Do-While Loop: Executes the task first, then checks the condition.
While Loop: Checks the condition before executing the task.
